{# HubL Syntax - HubL is the jinja inspired templating langauge #} {# TYPES OF DELIMETERS ============================================== #} {% %} - Statement delimeters {{ }} - Expression delimeters {# #} - Comment delimeters {# Statements #} {% %} Used to create: - Editable modules - Define conditional template logic - Set up loops - define variables *NOTE Do not print anything to the page {# Expression #} {{ }} Expressions print values stored in the context of the template Example usage: A variable must be defined as a statement but then a hubL expression would be used to print the variable {# Do tag #} {% do %} The 'do' tag works like a statement, and can be used to modify lists and dictionaries *NOTE When adding values to arrays, use .append() instead. When adding values to objects, use .update() instead. {% set navigation = ["Home", "About"] %} {% do navigation.append("Contact Us") %} {{navigation}} {# Arrays Output #} [Home, About, Contact Us] {% set book = {"name" : "Rocking HubL", "author" : "John Smith" }%} {% do book.update({"ebook" : "yes"}) %} {{book}} {# Objects/Dictionary Output #} {name=Rocking HubL, author=John Smith, ebook=yes} {# MODULES ============================================== #} {% moduel unique name path="@hubspot/text" label="Single Text Line" value="This is a single text line" %} {# Output #}
This is a single text line
